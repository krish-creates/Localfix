# 🛠️ LocalFix: Service Marketplace
A full-stack marketplace built with **Streamlit** and **SQLite**.

### 🌟 Features
- **User Portal**: Book technicians and chat in real-time.
- **Tech Portal**: Manage job requests and build a profile.
- **Admin Dashboard**: Approve or Revoke technicians with custom feedback.
- **Real-time Ratings**: Dynamic star ratings based on user reviews.

### 🚀 How to Run
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
